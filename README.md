**LeviafanLoad** — это десктопное WPF-приложение, созданное для профессионального сканлейта и подготовки глав. Благодаря встроенному движку WebView2, программа обходит большинство современных веб-защит (Cloudflare, хотлинк), успешно вытягивает изображения из `canvas` и справляется с ленивой загрузкой (lazy-load).

---

## ✨ Главные особенности

* 🛡️ **Обход защиты:** Скачивание происходит прямо из контекста встроенного браузера. Приложение имитирует поведение пользователя (умный скролл) и перехватывает трафик, что позволяет скачивать картинки с 90%+ защищенных сайтов.
* ♾️ **Извлечение из Canvas:** Умеет вытаскивать кадры, даже если корейская/японская читалка скрамблит изображения и отрисовывает их через HTML5 `<canvas>`.
* 📦 **Пакетная загрузка:** Укажите шаблон URL (например, `site.com/chapter-{i}`) и диапазон глав — приложение скачает всё автоматически, останавливаясь только при достижении конца (ошибка 404).
* ✂️ **Умная постобработка (Webtoon Mode):** Встроенный графический движок собирает разрозненные сканы разной ширины в один бесшовный рулон и аккуратно нарезает его на части заданной высоты (например, по 4000px).
* 🗜️ **Авто-архивация:** Моментальная упаковка готовой главы в `.zip` архив с очисткой временных файлов.
* 🔄 **OTA Обновления:** Встроенная система автоматической проверки и установки обновлений прямо из GitHub Releases.

## 🚀 Установка и запуск

1. Перейдите в раздел [Releases](../../releases/latest).
2. Скачайте последнюю версию архива `LeviafanLoad.zip`.
3. Распакуйте в удобную папку на вашем ПК.
4. Запустите `LeviafanLoad.exe`.

> **Требования:** Для работы программы на старых версиях Windows может потребоваться установленный [Microsoft Edge WebView2 Runtime](https://developer.microsoft.com/ru-ru/microsoft-edge/webview2/) и .NET Framework.

## 🛠️ Как использовать

### Одиночная загрузка
1. Откройте нужную главу во встроенном браузере.
2. В правой панели укажите папку для сохранения и выберите нужные параметры обработки (склейка, ZIP-архив).
3. Нажмите **«ЗАПУСТИТЬ ЗАГРУЗКУ»**. Программа сама прокрутит страницу, соберет файлы и обработает их.

### Пакетная загрузка (Несколько глав)
1. Включите галочку **«Пакетная загрузка»** в настройках.
2. Введите шаблон URL, заменив номер главы на `{i}` (например: `https://mangasite.com/read/title/chapter-{i}/`).
3. Укажите начальную и конечную главу (или включите режим **«Авто до ошибки»**).
4. Укажите шаблон названия архива (например: `Глава {i}`).
5. Запустите процесс и можете идти пить чай — программа всё сделает сама.

## 💻 Технический стек

* **C# 7.3 / WPF** — Основа и UI приложения.
* **Microsoft.Web.WebView2** — Браузерный движок на базе Chromium для рендеринга и внедрения JS-инъекций.
* **SixLabors.ImageSharp (v2.4.2)** — Быстрая обработка графики в фоновом потоке с минимальным потреблением оперативной памяти.
* **System.IO.Compression** — Работа с архивами.
